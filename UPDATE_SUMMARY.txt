## 📡 API Endpoints

### Existing Endpoint (Updated)
```
GET /api/search/
Query: ?category=<type>&q=<search>
```

### New Endpoint
```
GET /api/hidden-places/
Query: ?category=<type>&search=<query>&city=<city>
Response: Includes category statistics
```

**Sample Response:**
```json
{
  "count": 6,
  "city": "pollachi",
  "places": [...],
  "categories": {
    "hidden_spots": 6,
    "falls": 4,
    "temple": 4,
    "food": 4
  }
}
```

## 📝 Installation & Running

### First Time Setup
```bash
# Install dependencies (if needed)
pip install -r requirements.txt

# Apply migrations
python manage.py migrate

# Populate data
python add_sample_data.py

# Run server
python manage.py runserver
```

### Access the Application
- **Frontend:** http://localhost:8000/
- **API:** http://localhost:8000/api/hidden-places/
- **Admin:** http://localhost:8000/admin/

## 💡 Tips

- Use `/api/hidden-places/` for location queries (default: Pollachi)
- Use `/api/search/` for general searches across all fields
- Rating field is easily customizable per location
- Images can be added per location in Django admin
- All changes are backward compatible

## 📞 Support

If you need to:
